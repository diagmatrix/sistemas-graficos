# Geometría por Revolución
Mediante este ejercicio el alumno aprenderá a crear geometría por revolución a partir de
una línea definida en un plano.
El vídeo geometria-revolucion.mp4 muestra un posible resultado del ejercicio.
De nuevo, no es necesario que se puedan modificar las figuras interactivamente.
Basta con que el estudiante se familiarice con los distintos parámetros al crear
una figura por revolución.
